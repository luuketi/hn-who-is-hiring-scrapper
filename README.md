# HN Who is hiring Scrapper

A Clojure library designed to scrap **Ask HN: Who is hiring?**, filter and sort by salary.

## Usage
```
(main 31235968
      ["U.S. REMOTE" "US-based" "US only" "US-Only"]
      ["remote"]
      "/tmp/asdf.html")
```
